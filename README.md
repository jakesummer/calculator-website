# calculator-website
Live Site: https://jakesummer.github.io/calculator-website/

## Keyboard Use
* Use the number keys to type numbers
* Operator keys to perform math operations
    * `+` to add
    * `-` to subtract (use before you enter the number to make it negative)
    * `*` or `x` to multiply
    * `/` to divide
* `Enter` or `=` to get result
* `Backspace` to delete previous digit
* `c` to clear
* `.` for decimals
